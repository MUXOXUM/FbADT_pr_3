# Компании "Кассиопея"
Сайт для визуализации космических данных

---

## Переменные окружения (.env)

Создайте файл `.env` в корне проекта со значениями по подобию `.env.example`.

## Запуск проекта

### Использование Docker Compose

В корне проекта выполнить:

```bash
docker-compose up --build
```

Сайт будет доступен по `localhost:8080`
